Gutter and Gap

Gutters (values between 0-5) specifically adjust the margins and padding on the bootstrap grid row and col which effects the spacing between columns inside the row. Left and right paddings are set on each column, and negative margins are used to offset that at the start and end of each row.

  • gx-* classes can be used to control the horizontal gutter widths.
  • gy-* classes can be used to control the vertical gutter widths within a row when columns wrap to new lines.
  • g-* classes to control both.

default

Bootstrap containers and columns have default left and right paddings of 12px, while the rows have default left and right margins of -12px:
