Vertical margins change when they touch.
Horizontal margins never change.
Note: a zero size margin can also touch another margin.

Remember that margins are meant to increase the distance between siblings, not to increase the gap between a child and its parent's bounding box; that's what padding is for.

Vertical margins calculation

  1. If both are positive they collapse to the biggest.
  2. Negative and positive margins sum up.
  3. If both are negative they collapse to the most negative.

margin: 8px 0

margin: 24px 0

margin: -4px 0

margin: 24px 0

margin: -4px 0

margin: -4px 0

Nested elements with touching vertical margins

When vertical margins touch between nested elements, a margins calculation happens between them and the resulting margin is applied to the parent:

margin: 24px 0

margin: 24px 0

margin: 24px 0

After the resulting margins have been applied to the parents, the margins calculation can now happen between the adjacent siblings:

margin: 8px 0

margin: 24px 0

A padding, but also a border, can prevent the vertical margins from touching:

margin: 24px 0

margin: 24px 0

margin: 8px 0

margin: 24px 0
