  1. By default elements are placed on the stacking context of the <html> element which renders on layer 0 corresponding to z-index: 0.
  2. If several elements are placed on the same layer, the stacking order follows the order of appearance in the HTML source code.
  3. z-index only works on positioned elements or on flex and grid items. A z-index value creates a stacking context with a single element as the root element. When a new stacking context is formed, that stacking context confines all of its descendants to the z-index used to create the stacking context.
  4. Stacking contexts can be contained in other stacking contexts, and they can together create a hierarchy of stacking contexts.
  5. The CSS opacity property with a value less than 1 also creates a stacking context. Several other CSS properties, in addition to opacity, create stacking contexts. The created stacking context with all of its descendants are confined to z-index: 0.
