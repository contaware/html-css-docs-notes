Automatic minimum size of flex items

A flex item cannot be smaller than the size of its content along the main axis.

  • For flex-direction: row, a flex item has min-width: auto applied when overflow-x is visible. For overflow-x: hidden → min-width: 0
  • For flex-direction: column, a flex item has min-height: auto applied when overflow-y is visible. For overflow-y: hidden → min-height: 0

<img> style min-width: 0

<img> style overflow-x: hidden

<img> no style

With flex-direction: column an image behaves like having width: 100%

By default align-items is stretch, the image has no width set, so it fully stretches to its container and maintains the aspect ratio.
