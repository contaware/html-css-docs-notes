Flexbox flex

  • Item 2 has flex-grow: 1 and flex-shrink: 1 set.
  • Item 3 has flex-grow: 2 and flex-shrink: 2 set.
  • The default for flex-grow is 0 (no grow), positive values mean grow in the available space with that proportion.
  • The default for flex-shrink is 1 (shrink), positive values mean shrink to fit with that proportion.

Flexbox items order

Items with the same order value belong to the same ordering group. The default order value is 0. Items are placed in visual order according to their order value, lowest values first. If more than one item has the same order value, then within that group the items are laid out as per html order.
