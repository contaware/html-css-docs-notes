The flexible box layout is a one-dimensional model which determines the free space and grows/shrinks/distributes the items in that space.
Free space = Flex container size - Total flex-basis size of all flex items
(free space can be positive or negative)

Flexbox flex

  • flex-basis specifies the initial size of the flex items before the free space is distributed according to the flex factors. flex-basis: 100px overrides the item's width/height property. flex-basis: content means the content size, item's width/height propriety is ignored. Default for flex-basis is auto which means the item's width/height property if defined, otherwise the content size.
  • An item's min-width/min-height or a max-width/max-height will always limit the flexing.
  • Default for flex-grow is 0 (no grow), positive values mean grow in the available positive free space with the given proportion.
  • Default for flex-shrink is 1 (shrink), positive values mean reduce the negative free space with the given proportion.
  • Shorthand flex: flex-grow flex-shrink flex-basis;

In this example item 2 grows and shrinks with a factor of 1 and item 3 with a factor of 2.

Flexbox items order

Items with the same order value belong to the same ordering group. The default order value is 0. Items are placed in visual order according to their order value, lowest values first. If more than one item has the same order value, then within that group the items are laid out as per html order.

Enable flex wrap

  • Add flex-wrap: wrap to container.
  • align-content sets the spacing of the lines that are created when the items wrap.
  • align-items sets the items' alignment in relation to the line they currently belong to.
