Flex items' automatic minimum size

For flexbox items, the auto keyword is the initial value of min-width.

With overflow-x: visible (default) the automatic minimum width computes to the content-based minimum width:

1.HiFirstChildHiSecondOneHiThirdOne

With overflow-x: hidden the automatic minimum width computes to zero:

2.HiFirstChildHiSecondOneHiThirdOne

With min-width: 0 the minimum width is explicitly set to zero:

3.HiFirstChildHiSecondOneHiThirdOne

Note: the same holds for min-height and overflow-y.
