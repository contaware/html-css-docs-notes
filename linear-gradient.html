Linear gradient top to bottom (default)

linear-gradient(red, yellow)


Linear gradient with 0deg is bottom to top

linear-gradient(0deg, red, yellow)


Linear gradient with transparency

linear-gradient(90deg, rgba(0,0,0,1.0), rgba(0,255,255,0.1))


Linear gradient with multi-stops

linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet)


Linear gradient with double-stops

linear-gradient(to right, red 33.3%, green 33.3%, green 66.6%, blue 66.6%)
