rem vs em vs %

em is relative to the parent font-size, while rem (root em) is relative to the root font-size which is the font size of the html element (equivalent to the :root pseudo-class).

If there is no explicit font-size value set on the root element, it inherits its value from the Browser settings, and usually that's 16px. An absolute font-size value on the root element overwrites the Browser settings, not always a good idea. On the other hand, it's possible to set a rem font-size for the root element, it will be relative to the Browser settings.

When percentages are employed with font-size, it's the same as using em. But that holds only for font-size, for example using percentages on a width, means relative to the parent width, while em on a width, means relative to the parent font-size.
