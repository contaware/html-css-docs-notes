The <img>'s width and height attributes

Those attributes are the weakest kind of styling, they are overridden by any css styling information, even for selectors with very low specificity. Setting the width and height attributes is still a good idea, as they are fallback sizing and sizing information as the page is loading.

Is height: auto necessary for a responsive image?

On an image without a height dimension explicitly set anywhere, the default for height is auto. To have a responsive image providing max-width: 100%; height: auto is good practice because the second css property overwrites a possibly set height attribute:

Images for High DPI displays

  1. First possibility is to use the srcset attribute:
    <img src="images/image-1x.jpg" srcset="images/image-1x.jpg 1x,images/image-4x.jpg 4x" alt="image" width="100" height="220">
