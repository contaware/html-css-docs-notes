object-fit

The object-fit property specifies how the contents of a replaced element should be fitted to the box established by its used height and width. A normal image stretches to its width and height, that's object-fit: fill. With object-fit: contain or object-fit: cover it's possible to change how the image is rendered inside it's width x height box.

background-size

background-size: contain

background-size: 100% 100% (like object-fit: fill)

background-size: cover

Note: an unwanted side-effect of using a div element with background-size is that if the user tries to print your page, a lot of times the image won't be printed as it will be deemed a background by the browser and ignored.

object-position and background-position

They both are used to position the image inside the content box. The defaults are object-position: center and background-position left top.

If one value is provided the second value defaults to center.

If two values are provided:

  • The first value can be left, center, right or it can be the distance from the content box left edge.
  • The second value can be top, center, bottom or it can be the distance from the content box top edge.
  • If the first value is in percentage, it means that a vertical line X% from the left of the image coincides with a vertical line X% from the left of the content box
  • If the second value is in percentage, it means that a horizontal line Y% from the top of the image coincides to a horizontal line Y% from the top of the content box.

If four values are provided, then the first and the third values do name the edges from which to calculate the distances.
