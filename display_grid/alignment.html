Align grid items

  • justify-self aligns a grid item inside a cell along the row axis (align all at container level with justify-items).
  • align-self aligns a grid item inside a cell along the column axis (align all at container level with align-items).
  • place-self sets both align-self and justify-self in a single declaration and place-items sets both align-items and justify-items in a single declaration.

Align rows and columns in grid container

  • justify-content aligns the entire grid along the row axis.
  • align-content aligns the entire grid along the column axis.
  • Can use: start (default), end, center, stretch, space-around, space-between, space-evenly.
  • place-content sets both align-content and justify-content in a single declaration.
