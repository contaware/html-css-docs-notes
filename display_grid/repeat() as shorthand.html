The repeat() function as shorthand

repeat(4, 100px) with a count as the first parameter and a size as the second parameter:

100px
100px
100px
100px

repeat(2, 50px 150px) with a count as the first parameter and multiple sizes as the second parameter:

50px
150px
50px
150px

Can mix repeat() with values outside of it:

150px
50px
50px
150px
