Custom properties (=CSS vars) are left almost entirely unevaluated,
except that they allow and evaluate the var() function in their value

#container1 {width: 800px; --half-width: calc(100% / 2);}
#container2 {width: 400px;}
#test {width: 200px; margin-left: var(--half-width);}

var(--half-width) is evaluated here and the 100% is referring to the width of #container2
