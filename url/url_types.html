URL types

Absolute

https://www.example.com/
http://www.example.com/

Relative to scheme used by current page

//www.example.com/

If the browser is viewing this page with https://, then it will request the above link with https://, otherwise it will request it with http:// (or if you are viewing this page locally with file://, it will use that scheme).

Warning: it's now discouraged to use them.

Relative to current page location

Note: these are the only relative URL types which work also with the local file:// protocol.
