width: auto vs width: 100%

box-sizing: content-box

I'm the containing block.

I'm the first child element. My width is auto.

I'm the second child element. My width is 100% of my parent's containing width.

I'm the third child element. My 20% margin and 10% padding are relative to my parent's containing width (not height).
