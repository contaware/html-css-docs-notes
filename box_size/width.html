• The default for width is auto which grows as much as it can, but it takes padding, margin and border into account.
  • The width property with percent is calculated in two steps:
    1. The width is read from the element's containing block, which usually is the parent element. But what width is read? If the element has a position: static (the default) or a position: relative, then the width is read from the content-box of the containing block, but if the element has a position: absolute, then the width will be read from the padding-box of the containing block. Note: the margin and padding of an element in percentage are also based on this same read width, even for padding-top, padding-bottom, margin-top and margin-bottom.
    2. Now the above read width is applied to the element according to the element's box-sizing property (box-sizing is not inherited).
box-sizing: content-box;
width: 100%;

box-sizing: content-box;
width: 100%;
padding: 10px;

box-sizing: content-box;
width: 100%;
border-width: 10px;

box-sizing: content-box;
width: 100%;
padding: 10px;
border-width: 10px;
